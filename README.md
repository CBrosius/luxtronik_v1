# luxtronik_v1
Luxtronik_v1 ESPHome Custom Sensor

HowTo:
1. Place luxtronik_v1_sensor.h in your ESPHome-ConfigRoot
2. Edit luxtronik_v1.yaml according to your needs
3. Compile and Upload to your Sensor
